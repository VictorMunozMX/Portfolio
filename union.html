Project: Union Papelera of Mexico

This was my first project as a programmer after finishing university. The objective of this project was to migrate the already obsolete sales system. The migration was from the FoxPro to Visual Basic language.

The development was divided into four stages: analysis, design, implementation, and testing. The analysis, which I consider to be the most important aspect of any system, included all of the information, including inputs, processes, and outputs. In the design, all aspects of how to interact between the user and the system are planned. The implementation consists of placing the system at the point of sale to later go to the testing stage, where tests are carried out with the end user and the necessary changes are made.

Finally, the system was developed in visual basic and contains modules such as inputs, outputs, point of sale, accounting, payroll, cash, and it currently continues to work.
